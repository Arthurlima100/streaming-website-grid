# 🎬 Streaming Website Grid

Projeto desenvolvido para praticar **CSS Grid** e **Flexbox**, recriando o layout de uma página de streaming com base em um design do Figma.

## 🚀 Objetivo

O principal objetivo deste projeto foi treinar:

* Estruturação de layouts com **CSS Grid**
* Alinhamento e organização com **Flexbox**
* Interpretação de layout a partir do Figma
* Boas práticas de HTML e CSS

## 🖥️ Layout

O projeto consiste em:

* Um vídeo em destaque (seção principal)
* Uma sidebar com lista de vídeos
* Área de propaganda
* Informações como título, visualizações e descrição

## 🛠️ Tecnologias utilizadas

* HTML5
* CSS3
  * Grid
  * Flexbox

## 📌 Observações

* O layout **não é responsivo** (feito apenas para desktop)
* Este foi um projeto de prática, com foco em aprendizado
* Futuramente, pretendo melhorar:

  * Responsividade 📱
  * Ajustes de layout e espaçamento
  * Organização do código



## 📚 Aprendizados

Durante o desenvolvimento, reforcei conceitos importantes como:

* Diferença entre Grid e Flexbox
* Quando usar cada um
* Controle de espaçamento com `gap`
* Estruturação de layouts mais próximos do mercado
